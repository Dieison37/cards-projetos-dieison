Quiz-Rock

➤

App de Quiz interativo onde foi criado com HTML, Bootstrap para fazer as estilizações e o JavaScript com a lógica das perguntas e respostas, a nota é exibida em tela baseado na porcentagem de erros e acertos.

To-Do List

➤

É uma lista de tarefas melhorada que efetua a busca por caracteres e oculta quando não é encontrado, o javascript achei bem complexo da forma que foi inserido mais e bem funcional, os dados são salvos localmente no LocalStorage.

Weather App

➤

Essa é uma aplicação que busca por uma cidade mostrando o clima e a temperatura atual, as requisições são realizadas pela API accuweather.com. Uma aplicação simples mas que ao mesmo tempo complexa pelos detalhes no código, quando uma busca é concluída mostra um card relacionado com o contexto atual da região buscada.

Conversor de Moedas

➤

App que converte moedas, onde se integra a uma API que efetua as buscas das moedas e as converte em tempo real, Ex: BRL (Real) para USD (Dóllar) e vice-versa, mas tem várias outras moedas de vários países, o valor é atualizado em tempo real referente a moeda em questão e as requisições são obtidas através da ExchangeRate-API.com.

Dev Finances

➤

Aplicação de controle financeiro, onde é inserido os valores de entradas e saídas e uma breve descrição da despesa incluindo datas, efetua a soma e subtração conforme o usuário vai inserindo dados e os mesmos são salvos no LocalStorage do navegador.

Expert Notes

➤

Aplicação realizada no evento NLW-Expert da Rocketseat onde foi desenvolvido um App de criação de notas que converte áudio em texto através de uma API a caniuse.com, as tecnologias usadas: ReactJS, TypeScript, Vite, Tailwind CSS, Lucide, Radix UI e Sonner.
